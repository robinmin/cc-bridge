### Background

Task 0162 Phase 5 deferred item. Currently tools are provided as a flat list. Need a multi-stage filtering pipeline that applies policies at different levels: global defaults, per-workspace overrides, per-chat session overrides, and per-tool granular controls.

### Requirements

1. Define ToolPolicy interface with allow/deny/transform stages
2. Implement global policy layer (workspace-level defaults)
3. Implement per-chat policy layer (session-level overrides)
4. Implement per-tool policy layer (granular tool controls)
5. Pipeline composition: global -> per-chat -> per-tool evaluation order
6. Policy configuration via TOOLS.md or separate policy file
