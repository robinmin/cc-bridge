## 0184. markdown-to-fts-indexing

### Background

The FTS index must be rebuildable from markdown source. This task implements the indexing pipeline that reads all .memory/ files and populates the FTS5 table.

### Requirements

Implement indexer.ts that scans .memory/ directory, parses markdown files, extracts title and content, inserts into FTS5. Must handle all memory types (memory.md, daily logs, bank pages).
