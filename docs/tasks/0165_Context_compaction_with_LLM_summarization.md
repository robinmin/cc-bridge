## 0165. Context compaction with LLM summarization

### Background

Task 0162 Phase 5 deferred item. Long-running agent sessions accumulate context that eventually exceeds the context window. Need LLM-powered summarization to compact older conversation history while preserving key information.

### Requirements

1. Detect when context window usage exceeds threshold (e.g., 80%)
2. Identify compaction candidates (older messages, completed tool calls)
3. Use LLM to summarize compaction candidates into condensed form
4. Replace original messages with summary while preserving key facts
5. Track compaction history for debugging
6. Configurable compaction strategy (aggressive vs conservative)
7. Preserve system prompts and recent messages (never compact)
