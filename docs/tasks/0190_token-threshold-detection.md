## 0190. token-threshold-detection

### Background

Compaction triggers when context exceeds token threshold. This task implements token counting to detect when compaction should run.

### Requirements

Implement token counter using tiktoken or equivalent. Calculate total tokens in memory context. Trigger compaction when contextTokens > contextWindow - reserveTokens (default: 16KB reserve, 20KB recent).
